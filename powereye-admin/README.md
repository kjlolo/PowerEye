# Power Eye Admin UI

React + Vite admin UI for site metadata CRUD.

## Setup

1. Copy env template:

```bash
cp .env.example .env
```

2. Set values in `.env`:

- `VITE_API_BASE_URL`
- `VITE_API_TOKEN` (optional)

3. Install and run:

```bash
npm install
npm run dev
```

4. Build for deployment:

```bash
npm run build
```
